Vertex

My Own Programming Language

A new high-level stack-based assembly-like programming language interpreted by .NET Core.

pkg rajat_patwari.example

fn main() -> vd {
    ld.lt str "Hello, World!"
    call std.sio:writeln(str) -> vd

    ret
}

// Writes "Hello, World!" to standard output.
